# <i class="fa fa-usd"></i> スポンサー

JHipsterは資金を集めるために[OpenCollective](https://opencollective.com/generator-jhipster)を使っています。資金は透明性のある方法でプロジェクトの経費（このウェブサイトの運営など）に充てられます（[OpenCollectiveのドキュメントを参照](https://opencollective.com/how-it-works)）。あなたの寄付は、プロジェクトが存続し、成長するのに役立ちます。

後援者やスポンサーになるには、[OpenCollectiveページ](https://opencollective.com/generator-jhipster)からお願いします。
## 免責事項

これは寄付です。見返りとして商品やサービスは期待されません。それらの目的での返金要求は拒否されます。

当団体は、[規約](/association/)に記載されている通り、フランスの非営利団体であり、ビジネスパートナーシップには関心がありません。

## 後援者としての特典

支援者はプロジェクトに毎月2ドルを寄付します。

私たちはOpen Collective APIを利用して、メインページにバッカーを自動的に表示しています：

- [https://www.jhipster.tech](https://www.jhipster.tech) のフロントページの「バッカー」セクション（月間約11万ビュー）。
- GitHubのメインプロジェクトページの「バッカー」セクション（月間約1.5万ビュー）。

誰でも経費を申請できます。その経費がコミュニティの発展にとって意味のあるものであれば、コアコントリビューターたちによって私たちのOpenCollectivの台帳に「マージ」され、経費を申請した人へ償還されます。

資金はまず、プロジェクトの直接経費に充当されます。
[Open Collectiveのドキュメント](https://docs.opencollective.com)に記載されているように、あなたのお金がどのように使われるかは公開され、完全に透明です。
- [バグバウンディ](/bug-bounties/)
- ウェブサイトホスティング
- デベロッパーツール
- 法的保護

その後、以下の用途でも使用します。

- プロジェクトグッズの支払い
- ミートアップ料金の支払い
- 開発者の集まりのための費用：コアチームの会議のための旅費と食事代




- 開発者向けイベントの費用を支払う：コアチーム会議の交通費と食費

## 助けて！スポンサーなのにロゴが表示されません！

各スポンサーシップレベルについて、Open Collective APIを使用して、メインページにスポンサーを自動的に表示しています。そのため、ロゴが表示されない場合は、以下の点をご確認ください。

- 正しいスポンサーシップティアを選択しているか確認してください。これは、上記のティアのいずれかである必要があり、カスタムの貢献ではありません。
- Open Collectiveのプロフィールを確認し、ロゴがアップロードされていることを確認してください。これが、メインページにロゴを自動表示するために使用されるものです。
- 数分間お待ちください。Open Collective APIはキャッシュされているため、ロゴが表示されるまでに数分かかる場合があります。

これらの手順で解決しない場合は、Open Collectiveのサポートにお問い合わせください。当社は彼らのAPIを制御できないためです。彼らが修正する必要のあるバグや技術的な問題があるかもしれません。

また、このページの上部にある免責事項を覚えておいてください：「これは寄付です。見返りとして商品やサービスは期待されません。これらの目的での返金要求は拒否されます。」
